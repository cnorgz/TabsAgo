# TabsAGO Chrome Extension - Installation Guide

## Overview
TabsAGO is a Chrome extension that helps you organize and manage multiple Chrome tabs into a single, organized view with different viewing modes and organizational options.

## Installation Instructions

### Method 1: Load as Unpacked Extension (Recommended for Testing)

1. **Download the Extension**
   - Extract the `TabsAGO-Extension.zip` file to a folder on your computer
   - Note the location of the extracted folder

2. **Open Chrome Extensions Page**
   - Open Google Chrome
   - Navigate to `chrome://extensions/` in the address bar
   - OR go to Chrome menu (three dots) → More tools → Extensions

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner of the extensions page

4. **Load the Extension**
   - Click "Load unpacked" button
   - Navigate to and select the folder containing the extracted extension files
   - Click "Select Folder"

5. **Verify Installation**
   - The TabsAGO extension should now appear in your extensions list
   - You should see the TabsAGO icon in your Chrome toolbar

### Method 2: Drag and Drop (Alternative)

1. **Extract the Extension**
   - Extract the `TabsAGO-Extension.zip` file to a folder

2. **Drag to Extensions Page**
   - Open `chrome://extensions/`
   - Enable Developer mode
   - Drag the extracted folder directly onto the extensions page

## Using the Extension

### Accessing TabsAGO
- Click the TabsAGO icon in your Chrome toolbar
- The extension popup will open with the main interface

### Keyboard Shortcut
- **Windows/Linux**: `Ctrl+Shift+S`
- **Mac**: `Command+Shift+S`
- This shortcut captures selected tabs and sends them to TabsAGO

### Features
- **Tab Management**: Organize multiple tabs into groups
- **View Modes**: Switch between different viewing options
- **Export/Import**: Save and restore tab configurations
- **Session Safety**: Backup and restore tab sessions

## Permissions Explained

The extension requests the following permissions:
- **storage**: Save your tab configurations and settings
- **tabs**: Access tab information to organize them
- **contextMenus**: Add right-click menu options
- **sessions**: Access recently closed tabs and sessions
- **activeTab**: Interact with the currently active tab

## Troubleshooting

### Extension Not Loading
- Ensure Developer mode is enabled
- Check that all files were extracted correctly
- Try refreshing the extensions page

### Extension Not Working
- Check if the extension is enabled in the extensions page
- Try disabling and re-enabling the extension
- Check the browser console for any error messages

### Keyboard Shortcut Not Working
- Ensure no other extensions are using the same shortcut
- Check Chrome's keyboard shortcuts settings
- Try restarting Chrome
