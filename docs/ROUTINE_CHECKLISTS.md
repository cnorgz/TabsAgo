# Routine Checklists for TabsAGO Project

## Pre-Coding Checklist
- [ ] Documentation review complete
- [ ] Dependencies identified
- [ ] File structure impact assessed
- [ ] Security requirements reviewed
- [ ] Error handling strategy established

## Code Change Checklist
- [ ] Backup or branch created
- [ ] Affected files identified
- [ ] Code changes reviewed for error handling
- [ ] Documentation updated (if needed)
- [ ] Tests written/updated
- [ ] Code linted

## Post-Implementation Checklist
- [ ] Code review completed
- [ ] Testing completed (locally)
- [ ] Documentation updated
- [ ] Git commit prepared with a descriptive message
- [ ] Change log updated
